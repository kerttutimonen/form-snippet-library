Form Snippet Library

Input

Inputs are fields in forms where the user can insert his/her data.

type="email"

is used for inserting and accepting e-mail addresses

type="number"

is used for numbers

type="url"

is used for website URLs

type="color"

is used for choosing a different color

type="search"

gives an possibility to search something

Textarea

Textareas are part of forms, but located outside the form, also used for multiple lines and more text than inputs.

textarea id="message"

write a text

Select

The user can select the answer from multiple options.
